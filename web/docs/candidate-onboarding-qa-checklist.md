# Candidate Onboarding QA Checklist

## New Candidate

1. Log in with a candidate account that does not have a completed profile.
2. Confirm the account redirects to `/candidate/onboarding`.
3. Complete Personal, Skills, Location, Experience, and Review.
4. Refresh midway and confirm saved progress remains.
5. Submit from Review and confirm the dashboard opens.

## Skills

1. Choose a main category.
2. Confirm only relevant subcategory skills appear.
3. Select three skills.
4. Confirm a fourth skill cannot be selected or saved.
5. Remove one selected skill and select another.
6. Refresh and confirm saved skills remain.

## UAE Location

1. Select `UAE`.
2. Confirm emirates appear.
3. Select `Dubai`.
4. Save and refresh.
5. Confirm `Dubai` remains selected.

## India Location

1. Change country to `India`.
2. Confirm UAE emirate choices are cleared.
3. Select an Indian state.
4. Save and refresh.
5. Confirm the state remains selected.

## Contact Privacy

1. Change phone visibility.
2. Save.
3. Refresh.
4. Log out and back in.
5. Confirm the value remains unchanged.

## Existing Candidate

1. Log in with an already completed candidate account.
2. Confirm there is no onboarding redirect.
3. Edit the profile.
4. Confirm existing fields not edited remain intact.

## Profile Photo

1. Open Personal details or Edit Profile.
2. Upload a JPG, PNG, or WebP under 4 MB.
3. Confirm the preview appears after save.
4. Try an unsupported file type and confirm it is rejected safely.
